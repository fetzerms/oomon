The motd.txt is actually just a plain text file that is read whenever an oper requests to see the Message of the Day using the .motd command. The stock MOTD is as follows: 
    Thank you for using OOMon!

This is the default MOTD

The official OOMon web site is:
   http://oomon.sourceforge.net/

Please visit the official site to report bugs,
suggest new features, or download updates!
You can change the filename of the MOTD by specifying it in the M: line of the oomon.cf file.
